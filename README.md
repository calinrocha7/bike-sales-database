Sales Performance Analytics Dashboard

Overview
This project analyzes sales data to identify low-performing products, examine revenue performance, and investigate customer order history.

Using SQL, I performed data aggregation and joins to extract meaningful business insights, and the results were visualized in Tableau.

Tools used:
MySQL
Tableau Public
